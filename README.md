# Checklist

1. Setup SSH Keys
2. Install Git
3. Bootstrap

   ```
   $ ./bootstrap.sh
   ```

4. [Setup Nord Terminal Theme](https://github.com/arcticicestudio/nord-terminal-app)
5. Install Tmux Plugins

   ```
   $ tmux
   Ctrl-b I
   ```

6. Install Vim Plugs

   ```
   $ vim
   :PlugInstall
   ```
